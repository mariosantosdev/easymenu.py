Como Usar

Comandos

  • menu_nocolor.Menu([opções], [título do menu], [pergunta do menu]) - Cria um menu sem cores
  • menu.Menu([opções], [título do menu], [pergunta do menu]) - Cria um menu com cores

Exemplos

  • Não é preciso adicionar a opção Sair, ela sempre será adicionada por padrão.
Menu Sem Cor
from EasyMenu import menu_nocolor

option = {
  "Sim": "print('Hello, World!')",
  "Não": "exit()"
}

menu = menu_nocolor.Menu(option, 'Deseja continuar ?', 'Escolha uma opção: ')
eval(menu.display())

Menu Com Cor
from EasyMenu import menu

option = {
  "Sim": "print('Hello, World!')",
  "Não": "exit()"
}

menu = menu.Menu(option, 'Deseja continuar ?', 'Escolha uma opção: ')
eval(menu.display())

Executar função customizada
from EasyMenu import menu_nocolor

def customFunction():
  print('Até Mais...')
  exit()

option = {
  "Sim": "print('Hello, World!')",
  "Não": "customFunction()"
}

menu = menu_nocolor.Menu(option, 'Deseja continuar ?', 'Escolha uma opção: ')
eval(menu.display())
